## SETUP

- Fork this repo.
- Clone the forked repo
- Run `yarn` to install all the dependencies ( there's only [jest](https://jestjs.io/) )

## COMMANDS

- `yarn test` test the function
- `yarn test --watch` run the test-suite in watch mode
